# CareBot MediMate (CBMM) — Project Write-Up

## Project Inspiration

The motivation behind **CareBot MediMate (CBMM)** stems from the growing pressure on global healthcare systems. Most patients often face long wait times for initial triage, and post-clinic medication adherence remains a significant hurdle in chronic disease management. In rural areas, access to these resources becomes an even bigger problem. 

We aimed to solve two critical problems:
1.  **Triage Inefficiency**: Empowering patients with an AI-driven "first contact" that can assess the urgency of symptoms using established medical guidelines.
2.  **Medication Non-Adherence**: Providing a virtual assistant that helps patients track side effects and provides coach-like support for treatment regimens.

In the context of Generative AI, CBMM demonstrates how agentic frameworks can move beyond simple chat by integrating Retrieval-Augmented Generation (RAG) and state-aware process flows to provide specialized, high-stakes assistance.

## Technology Stack

### Languages
- **Python**: Powering the backend logic, AI orchestration, and RAG services.
- **JavaScript (React)**: Driving the interactive, real-time frontend user interface.

### Frameworks and Libraries
- **Railtracks**: Our core agentic framework, utilized for defining robust agent nodes and flows for Triage and Health Assistance.
- **FastAPI**: A high-performance web framework for the Python backend, supporting Server-Sent Events (SSE) for real-time typewriter effects.
- **React**: Used for building a responsive and dynamic single-page application.
- **IBM WatsonX AI SDK**: The primary interface for interacting with Llama 3 models.
- **NumPy**: Leveraged for efficient vector operations and cosine similarity in our RAG service.
- **Pydantic**: Ensuring structured data validation across the system.

### Platforms
- **IBM WatsonX.ai**: Our foundational AI platform for LLM inference (Llama 3-3-70b-instruct) and text embeddings.
- **IBM Db2**: Utilized for persistent storage of medical knowledge and conversation logs (with a robust local JSON fallback for portability).

### Tools
- **Node.js & npm**: For frontend dependency management and build processes.
- **Pip & Virtualenv**: For Python environment and package management.
- **Git**: For version control and deployment collaboration.

## Product Summary

**CareBot MediMate (CBMM)** is an AI-powered healthcare assistant designed to bridge the gap between initial symptoms and professional clinical consultation.

### Key Features:
-   **AI Triage Bot**: A specialized agent that listens to patient symptoms, performs semantic searches against medical guidelines (RAG), and determines triage urgency (Emergency, See Doctor Soon, or Home Care).
-   **Virtual Health Assistant**: A medication coach that tracks patient experiences with their treatments, flags potential side effects based on severity, and maintains a therapeutic conversation.


### User Experience:
The user interface is designed to be clean, accessible, and intuitive. Users can toggle between "Symptom Triage" and "Health Assistant" modes, receiving immediate, documented feedback that helps them make informed decisions about their health.
